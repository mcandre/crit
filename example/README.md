# DEMO APPS

# EXAMPLES

```console
$ hello
Hello World!
```

# REQUIREMENTS

* [Rust](https://www.rust-lang.org/en-US/) 1.64+
* [cross](https://crates.io/crates/cross) 0.2.5+
* [Docker](https://www.docker.com/) 20.10.23+
* [crit](https://github.com/mcandre/crit)

# BUILD & INSTALL

```console
$ cargo install --bins --path .
```

# UNINSTALL

```console
$ cargo uninstall demo
```

# PORT

```console
$ crit
```
